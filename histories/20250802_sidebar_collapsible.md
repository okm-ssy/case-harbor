# サイドバー折りたたみ機能の実装

## 実装方針

サイドバーにトグルボタンを追加し、デフォルトで閉じた状態で起動するようにUIを改善した。

## 実装内容

### Sidebar.tsx の修正

1. **状態管理追加**
   - `isCollapsed` state を追加 (デフォルト: `true`)
   - 幅とコンテンツ表示の制御を実装

2. **UI要素の追加**
   - `sidebar-toggle` ボタンを追加
   - 折りたたみ状態に応じてアイコン変更 (▶/◀)
   - `collapsed` クラスの条件付き追加

3. **レスポンシブ対応**
   - リサイザーは展開時のみ表示
   - コンテンツは折りたたみ時は非表示

### CSS スタイルの追加

1. **アニメーション**
   - `transition: width 0.3s ease` でスムーズな折りたたみ

2. **トグルボタン**
   - 右上に配置、ホバー効果付き
   - ダークテーマに適した配色

3. **折りたたみ状態**
   - 幅40pxの細いサイドバー
   - コンテンツは完全に非表示

## 利点

- 初期表示でメインコンテンツ領域が広く確保される
- 必要に応じてサイドバーを表示可能
- 直感的なトグル操作
